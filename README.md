# Stegastatter

#### Preview 
This project provides several Steganography and Steganalysis algorithms that I worte (based on known concepts). Using these algorithms, one can not only encrypt data - but hide it inside images. The algorithms are layered on a server-client system with a communication algorithm (on the application layer of the OSI model) that I developed and can be seen on other projects that are also structered with the same system.

#### The algorithms included
This project supports Embedding, Extracting, and Capacity calculations for the following algorithms:
1. LSB (Least Segnificant Bit)
2. BPCS (Bit-Plane Complexity Segmintation)

Additinally, this project supports the following Steganalysis algorithms:
1. Image difference calculation
2. Image bit-plane slicing
